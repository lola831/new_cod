The Cod Mountain Fish Company is a family ownerd fishing charter service operating out of Santa Cruz, California. They offer a range of different trips and services, now reflected accurately on their new website.

## Features

### Fish Reports and Captain Login

The Fish Report page is an essential eleemnt of any fishing charter service's website and details the events of daily excursions. In order to allow only the owner to log in and be able to add and edit fish reports, a discreet Captain login feature was added - the Captain need only add the credentials to the URL and will be automatically logged in.
